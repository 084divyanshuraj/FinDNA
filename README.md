## 📌 Overview

**FinDNA** is a web application that helps individuals understand and analyze their personal financial habits. Just like DNA carries the blueprint of life, FinDNA uncovers the hidden patterns and blueprints of your financial health — tracking spending, visualizing trends, and helping users make smarter money decisions.

Built under the pressure and excitement of the **SRUJANA 2026 Innovation Fest** (3–5 April 2026) at Sri Vasavi Polytechnic Engineering College, Vizag — this project is a testament to what a passionate team can ship in a hackathon sprint.

---

## ✨ Features

- 📊 **Finance Dashboard** — Get a bird's-eye view of your financial health at a glance
- 💸 **Expense Tracking** — Log and categorize your spending habits
- 📈 **Trend Analysis** — Visualize income vs. expenditure over time
- 🧠 **Smart Insights** — AI-powered suggestions based on your financial patterns
- 🔐 **Secure & Private** — Your financial data stays yours

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- Python (3.10+)
- npm or yarn

### 1. Clone the Repository

```bash
git clone https://github.com/084divyanshuraj/FinDNA.git
cd FinDNA
```

### 2. Setup the Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

The backend server will start at `http://localhost:8000`

### 3. Setup the Frontend

```bash
cd findna-web
npm install
npm run dev
```

The frontend will be available at `http://localhost:3000`
